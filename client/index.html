Slipchart
Spring 2017 Hackathon hack by James Echmalian
It's not on the ballot but you are welcome to write it in!
What is it?
A demonstration of how we can breathe more life into our charts. Slipchart adds a dimension to conventional 2D charts and lets the user slide along the third dimension without pausing to reload the page and generate a new chart.
The user can select a particular time value by operating the slide-bar. Try it.
Chart #1
How about some animation?
Press the "Once Through" button to animate the chart through the full range.
Press the "Play" button to launch the chart into continuous animation mode.
Chart #2
Just to prove that the above chart is not completely canned, here is another one based on the same component, but with different configuration and data.
At the techie level, this is a straight-up HTML5 browser component - just Canvas, no Flash, no charting package, not even jQuery.
If I had more time (or were a better hacker)... I'd implement export of animation to video for inclusion in presentations.
